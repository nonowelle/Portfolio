Planties

Planties is my first full fledged web application. It is inspired by the Yelpcamp project from The Web Developper Bootcamp course (by Colt Steele). Create an account and you will be able to collect and share photos of your plants with other enthusiasts. Thanks to this project, I learned how to use Express, Node.js and MongoDB.

Technologies used : Bootstrap - Express - Node.js - MongoDB - EJS

CRISM

As webmaster for the Canadian Substance Abuse Research Initiative (CRISM), my mandate was to proceed to a complete overhaul of their website. I had to use the Wordpress theme that was first implemented Jupiter 6. I created a new structure, established a new color palette and created mock-ups with Figma to finally implement the new design. I also added metadecriptions via the Yoast plugin to improve the natural referencing.

Technologies used : Wordpress - Thème Jupiter 6 - CSS - YoastSEO - Figma
